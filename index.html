.gitattributes Generator

Generates .gitattributes file for your project.

  • Binary Files: Input files that are truly binary.
  • CRLF Files: Input files that should have CRLF line endings on checkout.
  • LF Files: Input files that should have LF line endings on checkout.
  • Text Files: Input files that contains text only and don't have any special line ending requirements.

Note: Input Format - Comma seperated list of file extensions without prefix dot.

File will be available for download instantly. After downloading, rename the file to .gitattributes and put it into the root of your repository
